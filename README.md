# A Visual Analysis of Artificial Intelligence in Healthcare

The goal of the project is to analyze how artificial intelligence in medicine has evolved over time using bibliographic metadata from OpenAlex. 

The analysis focuses on publication growth, paradigm shifts, geographic distribution, medical–technical collaboration, and country collaboration networks.

## Dataset

The dataset was constructed using the OpenAlex Works API.

The processed dataset, together with the country collaboration network node and edge lists, can be downloaded from the following folder:
